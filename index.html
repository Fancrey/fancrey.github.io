Abstract

考虑凸集上的最小化矩阵秩问题，Rank Minimization Problem(RMP), 经常是NP-hard的。在一些特殊情况下，RMP包含了在凸集上寻找最稀疏向量的问题。

提出两种基于凸优化的方法逼近解决RMP. trace/nuclear norm 与 log-det. 两种方法适用于所有一般矩阵，并且很有效率，不需要初值，如果可行集有界，则可以在RMP上产生全局下界。 从最小化秩函数的凸包的意义上讲，nuclear norm heuristics 是最佳的，并且给出了理论支撑。在寻找稀疏向量的特殊情况下，这些启发式(heuristics)算法退化为l1−norm以及迭代l1−norm最小化方法。

Chapter 2 The Rank Minimization Problem

我们研究一般的矩阵秩最小化问题(Rank Minimization Problem RMP)，可表示为：

RMP:minimizeRankXsubject to X∈C

X∈Rm×n是优化变量，C是表示约束的凸集。例如：一个一般性的RMP，假设我们想要根据观测值以及先验假设信息估计或重建一个协方差矩阵：

X=E(z−Ez)(z−E)T

其中z∈Rn. 这里E(z)表示了随机变量 z 的期望。约束 X∈C 表示估算的协方差矩阵与我们的测量或观测数据以及先前的假设一致（或不可能）的条件。例如，X 中的元素应位于特定间隔内。X 的秩是衡量随机模型 z 复杂度的一种度量，某种意义上，它给出了解释 z 的协方差所需的独立随机变量的数量。因此，RMP(???) 是找到与观测值和先前假设一致的具有最小复杂度的随机模型(i.e.协方差)的问题。在其他问题的应用中，秩约束会有不同的解释，详见Section 2.2.

在(???)中，我们允许任意的在凸集上的约束。因此，我们涵盖了实践中出现的大量约束和规范。例如：对模型准确性或性能的约束很常见；e.g., f(X)≤t ,这里f(⋅)是描述性能的凸函数，t∈R是tolerance。下一个Section会给很多例子。

实际上，在模型复杂度(由Rank X衡量)与模型精度(由f(X)衡量)之间始终存在着权衡，通过求解以下RMP，我们可以得到权衡曲线

minimizeRank Xsubject to X∈Cf(X)≤t

2.1 Computational complexity

文献[20,92]介绍了RMP是NP-hard的，比如Boolean linear programming可以被表示为RMP：考虑是否存在 x∈Rm 满足Cx+d≥0 并且 xi∈{0,1} ，其中C∈Rn×m, d∈Rm (感觉不对) d∈Rn，表示为RMP的形式：

minmizeRank diag(x1,...,xm,1−x1,...,1−xm)subject toCx+d≥0

其中 diag(x1,...,xm,1−x1,...,1−xm) 的秩大于等于 m 当且仅当 xi∈{0,1} 时等号成立。

然而在一些RMP的特殊情况中，我们可以找到精确的解，e.g.，通过奇异值分解(SVD)求解。具体将在Chapter 4展开。

Multivariate statistical data analysis

二阶数据统计模型，例如Principal component analysis(PCA), factor analysis[59]从有噪声的数据中估计协方差矩阵。由于噪音的影响，估计的协方差矩阵是满秩的(以概率1)。因此，自然地，在这些方法中我们希望找到一个低秩的协方差矩阵。正如本章开始提及的，一个低秩的协方差矩阵使得数据模型变得简单，可理解。例如，考虑如下的因子分析问题：

minimizeRank(Σ)subject to   ||Σ−Σ^||F≤ϵΣ≥0Σ∈C

其中Σ∈Rn×n是优化变量，Σ^ 是待估的协方差矩阵，C 是表示 Σ 的先验假设或信息的凸集，||⋅||F 是矩阵的Frobenius范数。约束条件 ||Σ−Σ^||F≤ϵ 代表误差，e.g.，原始协方差矩阵与待估协方差矩阵的Frobenius范数必须小于等于给定的 ϵ . 约束中 Σ>0 保证了得到的矩阵是有效的协方差阵(协方差总要大于0)。在统计术语中，Rank(Σ) 对应着解释 Σ 的因子数量。

如果 C∈Rn×n (i.e.,没有先验信息)，这个问题有一种基于SVD的解析解；详见Section 4.1。然而，在 Σ 的元素中施加类似上下界的额外的约束条件将导致这一问题计算十分困难。

2.2.3 Quadratic and bilinear matrixes inequalities as rank constraints

二次及双线性不等式(BMI)在许多领域均很常见，尤其是控制和组合优化中是最重要的规则。在控制系统中，BMI作为许多NP-hard问题的通用框架，已经进行了广泛的研究。在组合优化中，由于非凸二次不等式可以捕获变量的布尔或整数约束，因此也是必不可少的。例如，约束条件 xi∈{−1,1} 与xi2≤1,xi2≥1 是等价的，然而第二个不等式是一个非凸的函数。

在这个Section，我们展示了如何将包含二次矩阵不等式和BMIs的优化问题转化为秩约束问题。

Bilinear Matrix Inequality (BMI) problems

考虑如下的二次矩阵不等式问题：

minimizecTxsubject toC+∑i=1nxiAi+∑i,j=1mxixjBij≤0

在控制理论中，通常更严格地限制双线性形式，这里我们将变量分裂为 x,y ，并且将约束条件替换为(BMI):

minmizecTx+bTysubject toD+∑i=1mxiAi+∑k=1tykBk+∑i=1m∑i,j=1mxiykCik≤0

BMIs包含了各种各样的控制问题，包括具有结构不确定性的综合，以及固定阶和固定结构的控制器设计。

我们将说明问题可以被转换为秩约束问题。首先将问题表示为：

第二个约束条件可以写为 W=xxT 。这个等式等价于：

Rank[WxxT1]=1

Rank[ABBTC]=Rank C+Rank (ABC−1BT)

Fast matrix computations（还有点问题）

低秩矩阵近似有时用于节省计算量。举一个简单的例子，假设我们想计算 y=Ax ,其中 A∈Rm×n ，对于 x 的各种取值，假设m,n取值很大。这要求 mn 次乘法。如果Rank A=r ，那么 A 可以表示为 A=RLT，其中R∈Rm×r,L∈Rn×r，(感觉这里又写错了)。因此，y=RLTx 可以仅使用 (m+n)r 次乘法进行计算。如果 r 远小于 m,n ，这将节省大量的计算资源。

最简单的矩阵近似问题可以表示为：

minimizeRank A^subject to ||A−A^||≤ϵ

其中 A^ 是优化变量，ϵ 是阈值。上述问题可以通过SVD解决，我们将在Chapter 4介绍。然而，当 A 具有特殊结构时(e.g. Hankel or Toelitz)，A^ 希望保留这种结构。这些新增的约束一般都会使问题的求解变得困难。

最近出现了一种应用在微光刻中的快速图像仿真技术。使用霍普金斯模型的光学图像计算涉及成像系统内核的双重卷积运算[38]。离散化之后，可以简化为对各种输入 x 及其调整后的值计算二次 y=xTWx 。对 W 使用低秩近似，称为最优相干分解（OCD），可以实现快速图像模拟器[96]。

2.3 The cardinality minimization problem

最小化向量 x∈Rn 的非零元素(即矩阵的cardinality，记为card x)是一种重要的RMP特殊形式。基数的其他常见术语包括权重，稀疏度和0-范数。0-范数可能会造成混淆，因为card x在通常意义上当然不是范数;它既不是一阶齐次的，也不是凸的。我们也会使用sparse去描述一个向量 x∈Rn 是大部分元素为0的，i.e. card x≪n 。

显而易见这个问题是RMP的特殊情况，令式(???)中X为对角矩阵，i.e. X=diag x .那么Rank X 与向量x 中非零元素的数量是等价的。约束条件 X∈C 退化为 x∈C¯ ，其中 C¯ 是在映射 x→diag x 下 C 的原像。在这种情况下，式(???)等价于在凸集上寻找最稀疏的向量，我们将它表述为Cardinality Minimization Problem(CMP):

minimizecard xsubject to x∈C¯

另一个例子是信号的小波分解，在将信号（例如图像）分解为已知基础信号（例如小波，边波，傅立叶）的线性组合时，稀疏的系数导致信号压缩。

CMP有时候也因为以下的原因被称为ℓ0-范数最小化问题。当 p≥1 时，标准的p-norm 定义如下：

||x||p=(∑i=1n|xi|p)1p

p-norm 最小化问题是最小化||x||p 并且满足 x∈C¯ 。这等价于最小化 p-norm 的 p 次幂，即：

minimize∑i=1n|xi|psubject to x∈C¯

limp→0∑i=1n|xi|p=card(x)

因此CMP可以看作为一个 p-norm最小化问题??? 在 p→0 的极限。???，??? 之间的相似性不会延伸得太远。对于p≥1 时，目标函数是凸的，因此通常可以有效率的求得全局最优解。对于 p<1时，目标函数非凸，会有大量的局部而非全局最小值。
